# HitAScreen 手動テスト計画

## 目的
OS 依存の挙動（グローバルホットキー、オーバーレイのフォーカス挙動、権限、モニタ座標整合）について、自動テストで担保しきれない箇所を手動で検証する。

## 結果に必須のメタ情報
各テストケースで以下を記録する:
- 実行日時
- macOS バージョン
- モニタ構成
- 権限状態（アクセシビリティ / 入力監視 / 画面収録）
- 手順
- 期待結果
- 実結果
- 合否
- 失敗時のログ/再現条件（失敗時）

## テストケース

### MT-01 常駐起動
- 手順:
1. アプリを起動する。
2. コントロールパネル（MainWindow）を閉じる。
3. トレイメニューからコントロールパネルを再表示する。
- 合格条件:
1. プロセスが継続し、終了しない。
2. トレイメニューが利用できる。
3. コントロールパネルを再表示できる。

### MT-02 ホットキー起動
- 手順:
1. 別アプリを前面にする。
2. `Cmd+Shift+M` を押す。
- 合格条件:
1. 体感遅延が目標（150〜300ms）以内でオーバーレイが表示される。
2. 前面アプリがクラッシュしない。

### MT-03 コンテキスト取得
- 手順:
1. セッションを起動する。
2. 診断パネルのログを確認する。
- 合格条件:
1. 診断ログに process/window/rect/display の情報が含まれる。
2. 取得失敗時はフォールバック理由が記録される。

### MT-04 ラベル操作（左クリック）
- 手順:
1. セッションを起動する。
2. 画面上のラベルを入力する。
- 合格条件:
1. マウス操作なしで対象要素がクリックされる。

### MT-05 右クリック/ダブルクリック
- 手順:
1. セッションを起動する。
2. `F2` を押してからラベルを選択する（右クリック）。
3. 再度セッションを起動し、`F3` を押してからラベルを選択する（ダブルクリック）。
- 合格条件:
1. 右クリックでコンテキストメニュー等が表示される（適用可能な場合）。
2. ダブルクリックが期待どおりに動作する。

### MT-06 キャンセル動作
- 手順:
1. セッションを起動する。
2. `Esc` を押す。
- 合格条件:
1. オーバーレイが即時に閉じる。
2. キーボード入力の介入が残らない。

### MT-07 モニタ切替
- 手順:
1. 2枚以上のディスプレイを接続する。
2. セッションを起動する。
3. 左右矢印キーを押す。
- 合格条件:
1. オーバーレイの対象ディスプレイが切り替わる。
2. 切り替え先でヒントが再生成される。

### MT-08 抑制（ゲーム向け）
- 手順:
1. 設定でフルスクリーン抑制を有効にする。
2. フルスクリーンアプリを前面にする。
3. ホットキーを押す。
4. フルスクリーンを終了して再試行する。
- 合格条件:
1. フルスクリーン中はホットキーが抑制される。
2. フルスクリーン終了後はホットキー登録が復帰し、セッション開始できる。

### MT-09 権限不足フォールバック
- 手順:
1. 必要権限のいずれかを未許可状態にする。
2. アプリ起動後、セッションを起動する。
- 合格条件:
1. アプリが落ちない。
2. 取得できる範囲の情報で処理が継続する。
3. 診断ログと権限表示で不足が分かる。

### MT-10 複数文字ラベル選択
- 手順:
1. 候補数を増やして 2 文字以上のラベルが生成される状態でセッション起動する。
2. 2 文字ラベルを最後まで入力する。
3. 3 文字ラベルを最後まで入力する。
- 合格条件:
1. 完全一致したラベルでのみ実行される。
2. 1 文字目で早期実行されない。

### MT-11 初回表示の位置合わせ
- 手順:
1. アプリを新規起動する。
2. 初回ホットキーを押し、ヒント位置が対象と揃っているか確認する。
3. オーバーレイを閉じて再度ホットキーを押す。
- 合格条件:
1. 初回と 2 回目以降で Y 方向のズレが無い。

### MT-12 ヘルプテキストの視認性（繰り返し）
- 手順:
1. オーバーレイを 5 回以上起動/終了する。
2. 各回で上部ヘルプテキストを確認する。
- 合格条件:
1. macOS メニューバーに隠れず、常に見える。

### MT-13 候補重なり/フィルタ
- 手順:
1. 親子構造がある UI 上でセッションを起動する。
2. ベースラインの重なり具合を確認する。
3. 候補フィルタ設定（role/depth）を変更して再起動する。
- 合格条件:
1. フィルタ後に重なりが減る。
2. 設定変更が表示に反映される。
