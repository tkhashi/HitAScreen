- 症状:
  - アプリ終了時に、複数アプリのフォーカスが高速で切り替わるような見た目で画面がチラつく。
- 調査対象:
  - オーバーレイの「閉じる/隠す」とフォーカス遷移の順序。
  - トレイ終了シーケンス（`overlay -> main window -> orchestrator -> tray`）。

- 合格条件:
  - 単一/複数ディスプレイ環境で通常終了しても、目視できるチラつきが発生しない。
